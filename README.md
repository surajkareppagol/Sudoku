# Sudoku

This is the `sudoku` game created in `C` language.

## How to play

There are 9 blocks and each block has 9 cells.

![Game board](https://raw.githubusercontent.com/surajkareppagol/Assets/main/29%20-%20Sudoku/gameBoard.png)
